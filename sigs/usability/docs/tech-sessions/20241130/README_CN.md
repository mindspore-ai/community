# 【精彩回顾】11月30日易用性SIG技术分享活动

2024年11月30日，MindSpore易用性SIG举行了第18次线上技术分享活动，主题是《基于昇思MindSpore的模型构建、大模型并行优化和打印模型结构的API实现》。

感谢开发者们的热情参与和各位老师的精彩分享！本次分享会的资料也给分享给大家，感兴趣的开发者们可以通过下方链接下载学习。
