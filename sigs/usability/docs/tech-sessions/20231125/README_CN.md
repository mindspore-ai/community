# 【精彩回顾】11月25日易用性SIG技术分享活动

2023年11月25日，MindSpore易用性SIG举行了第10次线上技术分享活动，主题是《数据处理开发调试和基于MindSpore的模型复现》。

感谢开发者们的热情参与和各位老师的精彩分享！本次分享会的资料也给分享给大家，感兴趣的开发者们可以通过下方链接下载学习。
