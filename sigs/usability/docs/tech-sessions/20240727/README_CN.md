# 【精彩回顾】07月27日易用性SIG技术分享活动

2024年07月27日，MindSpore易用性SIG举行了第15次线上技术分享活动，主题是《昇思MindSpore与Safetensors的对接和API的多态实现以及模型的精度优化和动态感知网络》。

感谢开发者们的热情参与和各位老师的精彩分享！本次分享会的资料也给分享给大家，感兴趣的开发者们可以通过下方链接下载学习。
